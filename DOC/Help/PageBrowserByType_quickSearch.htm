Быстрый поиск типов и объектов

Быстрый поиск позволяет найти объекты и типы по их названиям в дочерних ветках дерева выбранного каталога:

 

Для начала поиска введите одно или два слова через пробел и нажмите кнопку Enter на клавиатуре или иконку поиска в левой части строки.

Для очитки результатов поиска кликните на иконку с правой стороны строки или удалите все слова и нажмите Enter на клавиатуре

 

Например, поиск в дереве «каталога по типу»:

·         Поиск по одному слову будет производиться по имени объекта и по имени типа.

Например, поиск «срк» в каталоге «/Геофизическое оборудование/Аппаратура скважинная/АК/» не дат результатов

, а если попробовать найти «112» в этой же директории будут найдены два объекта с содержащие в названии эту строку

·         Поиск по двум словам будет производиться  “по имени объекта + по имени типа ” и в обратном порядке “ по имени типа + по имени объекта”.

Например, поиск «мак 83»в каталоге «/Геофизическое оборудование/Аппаратура скважинная/АК/» даст следующий результат совпадения
