3.0.8.0 ( 07.01.2020)

·         Реализована возможность выбирать несколько объектов или классов в проводнике

·         Реализована возможность выполнять действия для выбранного множества объектов

·         Реализована возможность открывать подробности для выбранного множества объектов

·         Реализована возможность удаления выбранного множества объектов

·         Отображение блокированных объектов в проводнике

·         Добавлен новый диалог выполнения действий

3.0.7.0 ( 17.02.2019)

·         Реализован экспорт истории в *.csv( чтоб можно было открывать в excel/calc)

·         Исправление отчёта: - добавлено поле "Дата создания объекта" в отчёт графика калибровок до выбранного дня. СТО Газпром георесурс 3.0-2016 приложение

3.0.6.3 ( 26.06.2018)

·         исправлены пути, обновлены WX3.1.1 для релизной сборки.

·         launcher обновлён и теперь не делает поиск распространяемых библиотек MSCPP: бинарник и укомплектован redist2013 и redist2017

·         добавлена инструкция для инженера (Ямбург)

·         SQL: изменение в логике удаления объектов, теперь для группы «ObjDesigner» допускается удалять только «свои» объекты, которые были созданы не более 7 суток назад

3.0.6.2 ( 20.03.2018)

·         исправлена ошибка при клике на свойство, которое не является ссылкой, не открывать её shell'ом

·         исправлена ошибка с «вылетом» при повторном открытии вкладки со списком отчётов.

·         SQL: Добавление скриптов проверки целостности истории и состояния объектов + скрипты реконструкции истории в случае позднего обнаружения ошибок при обновлении.

3.0.6.0 ( 18.03.2018)

·         Обновлена инструкция

·         сделан «каталог по местоположению» и объединён с «каталогом по типу», переключатель каталогов в начале строки местоположения

·         свойства типов и объектив указанные при создании как ссылки или файлы - подсвечиваются и открываются по клику, аналогично ссылкам в браузере

·         столбец типа/местоположения переключает каталоги, аналогично ссылкам в браузере, делает переход к местоположению или к типу

·         вывод WhatIsNew при обновлении версии (WhatIsNew так же выводится если не запускать программу 30 дней )

·         исправлена ошибка выделения при возврате по backspace

·         изменён алгоритм отображения ToolTip для элементов таблицы (теперь отображается через 1.5 секунды после наведения мыши на элемент)

·         исправлена ошибка обновления объектов после сброса поиска без группировки по типу

·         изменён алгоритм запроса поиска в деревьях по местоположению и по типу

3.0.5.1 ( 05.03.2018)

·         Добавлена общая инструкция

§  отображение свойств классов

o   Быстрый поиск по типу и/или объекту

o   Группировка объектов по типу

o   кнопка помощи с ссылкой на инструкцию

·         Исправление ошибок, оптимизация
